Privacy Policy

Last updated: March 2026

This Privacy Policy describes how the personal automation tools operated by David Jamieson ("I", "me", or "my") handle information when you use applications or scripts connected to this domain, including tools that interact with the TikTok, Instagram, and YouTube APIs.

1. Who This Applies To

These tools are personal automation scripts used solely by the account owner (David Jamieson) to manage and publish content to social media platforms. They are not public-facing applications and are not available for download or use by third parties.

2. Data I Collect

These tools do not collect, store, or process personal data from any third party. The only data handled is:

  • OAuth access tokens issued by TikTok, Instagram, and YouTube to the account owner. These tokens are stored locally on the account owner's device and are never transmitted to any server I operate.
  • Metadata about published videos (e.g. video title, upload timestamp, platform media ID), stored locally for deduplication purposes.
  • Publicly available Reddit post content (titles and story text) accessed via the Reddit public JSON API for the purpose of producing video content.

3. Data I Do Not Collect

I do not collect:

  • Personal information from any user other than the account owner
  • Cookies, analytics, or tracking data of any kind
  • Passwords or login credentials
  • Device identifiers, IP addresses, or location data

4. Third-Party Services

These tools interact with the following third-party APIs on behalf of the account owner:

  • TikTok for Developers — content publishing via the TikTok Content Posting API
  • Meta Graph API — content publishing via the Instagram Reels API
  • YouTube Data API v3 — content publishing via Google's YouTube API
  • Reddit Public API — reading publicly available story content

Each platform's own privacy policy governs how they handle data processed through their APIs.

5. Data Sharing

I do not share, sell, or disclose any data to third parties. No analytics services, advertising networks, or data brokers receive any information through these tools.

6. Data Retention

OAuth tokens are stored locally on the account owner's device and can be deleted at any time by removing the token files. Upload tracking records are also stored locally and are not transmitted anywhere.

7. Security

Credential files (client secrets, access tokens) are stored locally and excluded from version control via .gitignore. Access to these files is limited to the account owner's local machine.

8. Changes to This Policy

This policy may be updated from time to time. The "Last updated" date at the top of this page reflects when it was last revised.

9. Contact

If you have any questions about this Privacy Policy, you can reach me at david.jamieson.bz@gmail.com.
